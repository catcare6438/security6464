☁️ Supabase 雲端同步 推薦
⚠️ 尚未設定 Supabase，照片只存在本機
設定後所有人都能即時看到照片與貼文！
Step 1　到 supabase.com 免費註冊
Step 2　建立新專案（任意名稱）
Step 3　Storage → New bucket → 名稱填 wavemao → 勾選 Public
Step 4　SQL Editor 執行以下建表指令：
CREATE TABLE IF NOT EXISTS posts (id text primary key, data jsonb, created_at timestamptz default now()); Step 5　Settings → API → 複製 URL 和 anon key 填入下方
🐙 GitHub 照片共享設定
設定後，飼主上傳的照片所有人都能看到！
1. 到 github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token → 勾選 repo 權限 → 複製 token
3. 填入下方你的 GitHub 帳號、Repo 名稱、Token
